문법 - 태그(TAG): 설명하는 말 CSS: 스타일을 꾸밀 때 사용하는 스타일 시트 언어

2022.06.14 TUE / WEB1-11까지

& l t ; 여는 꺽새, & g t ; 닫는 꺽새
<img src="소스 출처.확장자">: 이미지 첨부, 꺽새 안에 height=, width= 순서 상관 없이 작성 가능
 이미지는 소스라는 추가 정보를 필요로 함 => 이미지 태그만으로는 첨부 불가
<li>: 리스트의 항목을 구분하는 자식 태그 (부모 태그가 없을 시 bullet-point 형식으로 작성됨)
<ul>: 순서 없는 리스트라는 뜻의 부모 태그(unordered list)
<ol>: 번호가 매겨질 리스트라는 뜻의 부모 태그(ordered list)
& nbsp; &emsp ;: 공백 넣기 (emsp가 살짝 더 긴 공백)


2022.06.15 WED / WEB1-15까지

<!doctype html>: 문서 형식이 html이라는 걸 표시하는 관용적 태그 ("이 페이지를 html로 표시해라"의 의미)
<html>: html 페이지의 시작과 끝을 의미
<head>: 웹 페이지의 외부 파일들을 링크할 때 사용. 본문 자체를 설명
<body>: 브라우저에 실제로 표시되는 내용
<title>: 페이지의 제목을 정하는 태그. 흔히 제목은 창 위쪽에 뜸
<meta charset="utf-8">: 페이지가 문자를 표시할 형식을 지정. utf-8이 가장 많이 쓰임 (charset = character set)
<a href="사이트주소">: 외부 링크로 연결 (a = anchor, href = hypertext reference)
 이 꺽새 안에 target="_blank": 새 탭에 열기
 이 꺽새 안에 title="링크를 눌러보세요": 커서를 댔을 때 설명이 뜨게 할 수 있음
  ex. 이 링크는 네이버입니다.
