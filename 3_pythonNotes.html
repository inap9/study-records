연산자

문자열
- 파이썬은 한 줄씩 컴파일링 해서 문자열이 다른 줄로 넘어가면 에러가 뜸
  문자열을 """ 혹은 '''로 만들면 공백문자 없이 다른 줄도 인식 가능
- 슬라이싱: a[n] 형식을 통해 문자열의 n+1번째(0부터 시작하기 때문에) 칸에 있는 문자를 출력 가능한데, (-n은 역순으로 출력)
  슬라이싱은 a[i:p:z] 형식을 통해 특정 인덱스부터 다른 인덱스까지의 문자들을 모두 출력 가능.
  여기서 i는 출력을 시작하는 인덱스, p는 멈추는 인덱스, z는 출력 간격을 의미한다. (i <= a < p)
  ex. I love you.를 a[0:7:2] 한다면 Ilv 출력. (I love you.)

출력
ex. a = 2000, b = 'Korea'
    "I have " + a + "dollars. I want to go to " + b 가 비효율적일 때,
    => "I have %d dollars. I want to go to %s." %(a, b) 형식으로 표현 가능 (%s로 쓰면 웬만한 거 다 가능)
- %d: 정수, %s: 문자열, %c: 문자 1개, %f: 실수
- 출력하는 자료형 대신 {}를 넣고 .format()에서 괄호 안에 값을 넣으면 {} 안에 원하는 값을 출력 가능
ex. "My name is {}".format("Hetia") >>> My name is Hetia.
    "My name is {n} and I am {a} years old".format(a = 13, n = "Hetia") >>> My name is Hetia and I am 13 years old
- %s에서 %와 s 사이에 숫자 삽입으로 간격 조정. 양수 삽입 시 문자열 좌측에 공백, 음수 삽입 시 우측에 공백.
- %f에서 %와 f 사이에 0.n을 넣으면 소수점 n의 자리까지 실수 출력, 여기서 0은 출력 간격

문자열 함수
a.count('b') => 문자열 a에서 b의 개수 세기, a.find(b) => 문자열 a에서 처음 b가 몇 번째 인덱스에 있는지 찾기 (없으면 -1 리턴), a.index()
a = "," a.join("abcd") >>> a,b,c,d
a.strip() => 공백 지우기, a.upper(), a.lower(), a.replace("abc", "def") => abc를 def로 교체
a.split(":") => :를 기준으로 문자열을 잘라서 리스트 생성

리스트
- 여러 개의 데이터를 변수 하나로 관리, 선언은 a = [b, c, d]
- 숫자와 문자를 섞어서 사용도 가능하고, 리스트 안에 리스트 포함 가능
- 문자열처럼 리스트 슬라이싱도 가능
- 리스트 두 개를 더하면 리스트가 하나로 묶여짐 (데이터를 합쳐서 더 길어짐)
- a[0:n] = [] 형식으로 리스트 항목 삭제도 가능 혹은 del a[0:n]
- sort(), append(), reverse(), index()
- a.insert(0, 3) => 리스트 a의 0번째 인덱스에 3을 삽입한다는 의미
- a.remove(15) => 리스트 a에서 15라는 값을 가진 처음 항목을 제거. 인덱스가 아닌 "값"을 입력
- a.pop(3) => 리스트 a에서 3을 출력 후 기존 리스트에서 3을 제거
- a.extend(i,p) => 리스트 a의 끝에 i와 p를 추가 (확장)

딕셔너리: 해시(Hash) 혹은 연관 배열(Associative Array), "Key를 통해서 Value를 얻는다" => 키 중복되지 않게 주의
    - key 부분에는 int, string, tuple이 들어갈 수 있는 반면 value 부분에는 아무거나 들어갈 수 있다.
    - info.keys(), info.values(), info.items(), info.clear(), info.get(찾을 키, 없으면 return할 값)
    - 새로운 키 추가는 info['nickame'] = 'Hetty', 키 삭제는 del info['name'] (순서가 없고 키만 있기 때문에 인덱스 사용X)
    ex. info = { 'name': 'Hetia', 'age': 13, 0: [1, 2, 3]}, 이름만 출력 시 print(dic['name'])


- 집합: 정해진 순서 없이 반복되지 않는 요소들의 모음으로 중복되는 데이터를 없앨 때 많이 사용한다.
    ex. s = set[(1, 2, 3)]과 s = {1, 2, 3}은 같다.
    - 집합 s1 = {1, 2, 3}, s2 = {2, 3, 4}에 대하여 교집합은 s1 & s2 혹은 s1.intersection(s2)를 통해 {2, 3} return 가능
    - " 합집합은 s1 | s2 혹은 a.union(b)
    - " 차집합 s1은 s1 - s2 혹은 s1.difference(s2), s2는 s2 - s1 혹은 s2.differnece(s1)
    - s.add(): 원소 하나 추가, s.update(): 원소 여러 개 추가
    - s.remove()


- bool: 참/거짓
    - 파이썬에서는 데이터가 있으면(1, "Hi", (2,3)) 참, 없으면(ex. None, "", [], 0) 거짓이라는 속성을 가짐.


- 변수: 값보다는 자료의 주소를 담는다고 알고 있으면 됨
  ex. a = 3에서, 메모리에서 3이라는 정수형 객체를 담는 주소를 a가 갖고 있음
   따라서 b가 리스트일 때, c = b를 한 후 b의 값을 변경하면 c의 값도 바뀐다는 것
   => b의 값은 변하되 c의 값은 기존 b의 값을 유지하고 싶을 땐, c = b[:] 아무것도 넣지 않은 슬라이싱으로 주소가 아닌 값만 전달
   => 혹은, from copy import copy 를 통해 copy 모듈을 가져온 후 c = copy(b)로 값만 전달
  - 변수 값 할당은 a, b = (1, 2) 혹은 (a, b) = (1, 2) 혹은 [a, b] = [1, 2] 혹은 a, b = [1, 2]로도 가능
  - 변수 값 switch는 a, b = b, a 를 하면 a와 b의 값이 쉽게 서로 바뀜!!
